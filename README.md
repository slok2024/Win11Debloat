这个项目是一个针对 Windows 11 系统的硬核深度优化与预装精简（Debloat）工具，它基于国外非常流行的开源脚本项目 Win11Debloat 进行了满配全功能的图形化（GUI）原生复刻。

简单来说，它的核心价值在于：把原本复杂的、需要敲命令行或跑 PowerShell 脚本的底层系统优化，变成了一款可以“一键勾选、傻瓜式执行”的桌面客户端。

🚀 核心功能板块
这个项目100%不删减地复刻了原版脚本的全部微调矩阵，主要分为以下几个核心部分：

📦 预装 UWP 矩阵（应用大扫除）

日常基础： 卸载自带的且不常用的“反馈中心”、“获取帮助”、“天气”、“地图”等微软全家桶。

流媒体与第三方捆绑： 彻底清除系统安装后自带的第三方“毒瘤”推广（如 Facebook、TikTok、Spotify、休闲糖果游戏等）。

OEM 残余全家桶： 完整覆盖了惠普 (HP)、戴尔 (Dell)、联想 (Lenovo) 等品牌机出厂自带的随机管理和监控随行服务。

高危隔离区： 额外提供针对 Edge 浏览器和微软应用商店（Microsoft Store）的卸载选项（默认安全隐藏或警告，防止误删）。

🔒 隐私与 AI 控制（数据断流）

切断遥测： 彻底禁用 Windows 系统的诊断数据收集、目标广告跟踪、本地搜索历史和位置追踪，还你干净、私密的系统环境。

AI 特性剥离： 一键禁用或彻底剥离系统底层的 Microsoft Copilot、Windows Recall（回忆快照）、Click to Do，以及新版画图/记事本中的 AI 创作层，大幅释放后台 CPU 和内存。

推荐广告隔离： 屏蔽开始菜单广告、锁屏“聚焦”提示、Edge 侧边栏及 MSN 资讯流。

⚙️ 系统与更新微调（硬核调校）

经典恢复： 一键恢复 Windows 10 那种高效、展开全布局的传统右键菜单。

体验优化： 显示已知文件名的扩展名、显示隐藏文件、关闭鼠标加速、禁用连按 5 次的粘滞键弹窗。

高级策略： 拦截并阻止 BitLocker 自动设备加密（防止丢密钥导致数据丢失）、关闭传递优化（禁止系统在后台给别人上传补丁占宽带）。

🖥️ 开始菜单与任务栏（纯净视觉）

任务栏清爽： 隐藏烦人的“小组件”按钮、“聊天 (Chat)”图标、“任务视图”按钮。

操作增强： 任务栏图标右键直接增加“结束任务”选项；支持修改搜索栏样式和多屏任务栏合并模式。

🛠️ 技术亮点与设计哲学
极致顺滑的纯原生 Tkinter 构建：
项目完全基于 Python 自带的原生 GUI 库（Tkinter/Ttk）进行编写，不需要安装任何第三方花哨的界面库，保证了程序极其轻量、启动几乎秒开，且绝对不会引入额外的 DLL 依赖报错。

高频防重绘引擎（高效过滤）：
右上角自带“🔍 过滤”输入框。内置了极其硬核的渲染锁机制，输入关键字时能在一瞬间动态隐藏或显示数以百计的复选框，即便在老旧机器或 legacy 环境（如 Win7/Win10）下也绝不卡顿、不耗费 CPU。

安全的底层管线调度：
点击“全面执行所选调整”后，工具会动态将用户的勾选转换为纯正的 PowerShell 指令流，通过隐藏窗口在后台静默注入系统注册表或调度 AppxPackage 卸载引擎，既专业又安全。

嫌 Win11 各种后台 AI 服务、遥测服务占用内存，想帮老机器深度瘦身、提升游戏帧率的电脑硬件爱好者。
